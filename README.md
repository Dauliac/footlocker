# Foot locker

my personal i3lock fork fork.

## Dependencies

first install :

- i3lock-color
- scrot
- imagemagick

## Installation

git clone and move footlocker in `/usr/bin/footlocker`

```bash
git clone git@github.com:Dauliac/footlocker.git && \
cd footlocker \
mv footlocker /usr/bin/footlocker
```



## Usage

simply :

```bash
footlocker
```
